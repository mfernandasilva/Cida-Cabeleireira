# Cida Cabeleireira - Landing Page

Landing page sofisticada e responsiva desenvolvida para o salão Cida Cabeleireira, focada em experiência do usuário (UX) e conversão.

## Sobre o Projeto

Este projeto é uma vitrine digital para uma profissional com mais de 25 anos de tradição. O design foi pensado para transmitir **confiança, elegância e modernidade**, utilizando uma paleta de cores sóbria com toques de verde oliva (brand color) e tipografia clássica.

### [Acesse o Projeto Ao Vivo](https://cida-cabeleireira.vercel.app/)

## Tecnologias Utilizadas

O projeto foi construído Next.js, uma tecnologia do ecossistema Web para garantir performance e SEO:

* **Next.js 16.1.6** (App Router)
* **CSS3 Custom Properties** (Sistema de cores e variáveis)
* **Google Maps API** (Integração de localização)

---

## Funcionalidades e Seções

* **Hero Section:** Apresentação com foco na essência da beleza.
* **Serviços:** Grid organizado com as especialidades do salão.
* **CTA:** Uma chamada para ação. 
* **Sobre:** Narrativa sobre a trajetória e tradição de 25 anos.
* **Localização Inteligente:** * Borda animada com efeito de "fogo/brilho" usando CSS Conic-Gradient.
    * Badge dinâmico de **Aberto/Fechado** baseado no horário real.
    * Integração interativa com Google Maps.
* **Footer:** Navegação rápida e CTAs diretos para WhatsApp e Telefone Fixo.

## Como rodar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/mfernandasilva/Cida-Cabeleireira.git
   ```
2. Instale as dependências:
    ```bash
    npm install
    ```
3. Inicie o servidor de Desenvolvimento: 
    ```bash
    npm run dev
    ```
4. Abra http://localhost:3000 no navegador
